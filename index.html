Introduction to Chart.js

Chart.js is a popular JavaScript library that allows developers to create beautiful, responsive, and interactive charts for web applications. It uses HTML5 Canvas for rendering charts, making them highly customizable and performant.

Why Choose Chart.js?

  • Simple yet flexible API
  • 8 chart types with responsive options
  • Built-in animations
  • Supports modern browsers and IE11+

Latest Version

4.4.0 Release Notes

Getting Started

To use Chart.js, you need to include the library in your project:

HTML 
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

Or install via npm:

Terminal 
npm install chart.js

Basic Structure

Creating a chart with Chart.js involves these basic steps:

  1. Create a canvas element in your HTML
  2. Get the canvas context using JavaScript
  3. Create a new Chart instance with configuration
HTML 
<canvas id="myChart"></canvas>
JavaScript 
// Get the canvas element
const ctx = document.getElementById('myChart').getContext('2d');

// Create a new chart
const myChart = new Chart(ctx, {
    type: 'bar',  // Chart type
    data: {
        labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'],
        datasets: [{
            label: '# of Votes',
            data: [12, 19, 3, 5, 2, 3],
            backgroundColor: [
                'rgba(255, 99, 132, 0.2)',
                'rgba(54, 162, 235, 0.2)',
                'rgba(255, 206, 86, 0.2)',
                'rgba(75, 192, 192, 0.2)',
                'rgba(153, 102, 255, 0.2)',
                'rgba(255, 159, 64, 0.2)'
            ],
            borderColor: [
                'rgba(255, 99, 132, 1)',
                'rgba(54, 162, 235, 1)',
                'rgba(255, 206, 86, 1)',
                'rgba(75, 192, 192, 1)',
                'rgba(153, 102, 255, 1)',
                'rgba(255, 159, 64, 1)'
            ],
            borderWidth: 1
        }]
    },
    options: {
        scales: {
            y: {
                beginAtZero: true
            }
        }
    }
});

Line Charts

Line charts are perfect for showing trends over time or comparing multiple data series. They're one of the most commonly used chart types for visualizing continuous data.

Basic Line Chart

Multi-line Chart

Code Example: Basic Line Chart

JavaScript 
const ctx = document.getElementById('basic-line-chart').getContext('2d');
const basicLineChart = new Chart(ctx, {
    type: 'line',
    data: {
        labels: ['January', 'February', 'March', 'April', 'May', 'June', 'July'],
        datasets: [{
            label: 'Monthly Sales',
            data: [65, 59, 80, 81, 56, 55, 40],
            fill: false,
            borderColor: 'rgb(75, 192, 192)',
            tension: 0.1
        }]
    },
    options: {
        responsive: true,
        plugins: {
            title: {
                display: true,
                text: 'Monthly Sales Data'
            }
        }
    }
});

Key configuration options for line charts:

fill
Whether to fill the area under the line
tension
Controls the line curvature (0 for straight lines)
pointRadius
Size of the data points
borderDash
Creates dashed lines [dashLength, gapLength]
stepped
Creates a stepped line (true/false or 'before'/'after'/'middle')

Stacked Bar Chart

Stacked Bar Chart

Stacked Bar Charts

Stacked bar charts allow you to show multiple data series stacked on top of each other. This is useful for showing part-to-whole relationships while still maintaining the ability to compare total amounts across categories.

To create a stacked bar chart, set stacked: true in the scale options.

Pie Charts

Pie charts are circular charts divided into segments, where each segment represents a proportion of the whole. They're best used when you want to show the composition of something.

Pie Chart

When to Use Pie Charts

When you have a small number of categories (ideally 5 or fewer)
When the parts add up to a meaningful whole (100%)
When you want to emphasize a significant segment
Hard to compare sizes of segments accurately
Becomes cluttered with too many segments
Not ideal for showing changes over time

Code Example: Pie Chart

JavaScript 
const ctx = document.getElementById('pie-chart').getContext('2d');
const pieChart = new Chart(ctx, {
    type: 'pie',
    data: {
        labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple'],
        datasets: [{
            label: 'Dataset 1',
            data: [12, 19, 3, 5, 2],
            backgroundColor: [
                'rgba(255, 99, 132, 0.8)',
                'rgba(54, 162, 235, 0.8)',
                'rgba(255, 206, 86, 0.8)',
                'rgba(75, 192, 192, 0.8)',
                'rgba(153, 102, 255, 0.8)'
            ],
            hoverOffset: 4
        }]
    },
    options: {
        responsive: true,
        plugins: {
            legend: {
                position: 'top',
            },
            title: {
                display: true,
                text: 'Color Preferences Survey'
            }
        }
    }
});

Doughnut Charts

Doughnut charts are similar to pie charts, but with a hole in the center. This can make them more visually appealing and allows for additional information to be displayed in the center.

Doughnut Chart

Customizing Doughnut Charts

cutout
Controls the size of the center hole (percentage of chart radius)
rotation
Rotation of the chart in degrees (default is 0)
circumference
Sweep of the chart (default is 360 degrees)

You can create semi-circle or gauge charts by adjusting the rotation and circumference.

70%

Code Example: Doughnut Chart

JavaScript 
const ctx = document.getElementById('doughnut-chart').getContext('2d');
const doughnutChart = new Chart(ctx, {
    type: 'doughnut',
    data: {
        labels: ['Desktop', 'Tablet', 'Mobile', 'Other'],
        datasets: [{
            label: 'Website Traffic',
            data: [45, 12, 38, 5],
            backgroundColor: [
                'rgba(54, 162, 235, 0.8)',
                'rgba(255, 206, 86, 0.8)',
                'rgba(255, 99, 132, 0.8)',
                'rgba(75, 192, 192, 0.8)'
            ],
            hoverOffset: 4
        }]
    },
    options: {
        responsive: true,
        plugins: {
            legend: {
                position: 'top',
            },
            title: {
                display: true,
                text: 'Website Traffic by Device'
            }
        },
        cutout: '70%'  // Controls the size of the hole
    }
});

Radar Charts

Radar charts (also known as spider or web charts) display multivariate data on axes starting from the same point. They're useful for comparing multiple variables and seeing which variables have similar values.

Radar Chart

When to Use Radar Charts

When comparing multiple variables with the same scale
When looking for patterns or outliers across multiple dimensions
When comparing two or more data series across multiple variables

Best Practices

Limit the number of variables (axes) to maintain readability
Use consistent scales across all axes
Consider using fill with transparency to show area

Code Example: Radar Chart

JavaScript 
const ctx = document.getElementById('radar-chart').getContext('2d');
const radarChart = new Chart(ctx, {
    type: 'radar',
    data: {
        labels: [
            'Eating',
            'Drinking',
            'Sleeping',
            'Designing',
            'Coding',
            'Cycling',
            'Running'
        ],
        datasets: [{
            label: 'Person A',
            data: [65, 59, 90, 81, 56, 55, 40],
            fill: true,
            backgroundColor: 'rgba(255, 99, 132, 0.2)',
            borderColor: 'rgb(255, 99, 132)',
            pointBackgroundColor: 'rgb(255, 99, 132)',
            pointBorderColor: '#fff',
            pointHoverBackgroundColor: '#fff',
            pointHoverBorderColor: 'rgb(255, 99, 132)'
        }, {
            label: 'Person B',
            data: [28, 48, 40, 19, 96, 27, 100],
            fill: true,
            backgroundColor: 'rgba(54, 162, 235, 0.2)',
            borderColor: 'rgb(54, 162, 235)',
            pointBackgroundColor: 'rgb(54, 162, 235)',
            pointBorderColor: '#fff',
            pointHoverBackgroundColor: '#fff',
            pointHoverBorderColor: 'rgb(54, 162, 235)'
        }]
    },
    options: {
        elements: {
            line: {
                borderWidth: 3
            }
        }
    }
});

Advanced Chart.js Features

Mixed Chart Types

Chart.js allows you to combine different chart types in a single chart. For example, you can have a line chart overlaid on a bar chart.

Mixed Chart

JavaScript 
const ctx = document.getElementById('mixed-chart').getContext('2d');
const mixedChart = new Chart(ctx, {
    type: 'bar',
    data: {
        labels: ['January', 'February', 'March', 'April', 'May', 'June'],
        datasets: [{
            type: 'bar',
            label: 'Revenue',
            data: [50, 60, 70, 180, 190, 220],
            backgroundColor: 'rgba(54, 162, 235, 0.5)',
            borderColor: 'rgb(54, 162, 235)',
            borderWidth: 1
        }, {
            type: 'line',
            label: 'Profit',
            data: [20, 25, 30, 60, 65, 75],
            fill: false,
            borderColor: 'rgb(255, 99, 132)',
            tension: 0.1
        }]
    },
    options: {
        scales: {
            y: {
                beginAtZero: true
            }
        }
    }
});

Animations and Transitions

Chart.js provides powerful animation capabilities that can make your charts more engaging. You can customize the animation duration, easing function, and even create custom animations.

JavaScript 
// Animation configuration example
const chart = new Chart(ctx, {
    // ... chart configuration ...
    options: {
        animation: {
            duration: 2000,  // Animation duration in milliseconds
            easing: 'easeOutQuart',  // Easing function
            // Custom animations for specific elements
            animations: {
                y: {
                    duration: 2000,
                    delay: 500
                }
            }
        }
    }
});

Plugins and Extensions

Chart.js has a plugin system that allows you to extend its functionality. You can create custom plugins or use community-developed plugins for features like:

Data labels
Zoom and pan
Advanced tooltips
Annotations
JavaScript 
// Example of using a plugin (datalabels)
const chart = new Chart(ctx, {
    plugins: [ChartDataLabels],  // Register the plugin
    // ... chart configuration ...
    options: {
        plugins: {
            datalabels: {
                color: '#fff',
                font: {
                    weight: 'bold'
                }
            }
        }
    }
});

Responsive Charts

Chart.js is responsive by default, but you can customize how charts respond to different screen sizes.

JavaScript 
const chart = new Chart(ctx, {
    // ... chart configuration ...
    options: {
        responsive: true,
        maintainAspectRatio: false,  // Allow the chart to not maintain aspect ratio
        aspectRatio: 2,  // Set a specific aspect ratio
        
        // Responsive options based on screen size
        plugins: {
            legend: {
                display: true,
                position: 'top',
                labels: {
                    // Make the text smaller on small screens
                    font: {
                        size: (context) => {
                            const width = context.chart.width;
                            return width < 600 ? 10 : 14;
                        }
                    }
                }
            }
        }
    }
});
